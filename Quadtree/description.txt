Вариант 23 ***

Написать программу для работы по запросам оператора с упорядоченной таблицей в двух ключевом ограниченном пространстве, реализованной в виде Q-дерева поиска.

Ключи - целые числа. Информация - строка произвольной длины.

Узел дерева содержит границы ключей в каждом пространстве, массив указателей на поддеревья и указатель на начало списка элементов (ключи + информация) в порядке возрастания ключей (приоритет по первому ключу). Данные хранятся только в листьях дерева.

В таблице не могут храниться записи с одинаковыми ключами. Количество элементов в одном квадрате ограничено числом N.

Предусмотреть следующие операции:
- включение нового элемента в таблицу без нарушения свойств упорядоченности; если информация с заданным ключом уже есть, то выводится сообщение об ошибке;
- удаление из таблицы элемента, заданного своим ключом, без нарушения свойств упорядоченности таблицы; при уменьшении количества элементов в квадрате квадраты не удаляются;
- поиск информации по заданной паре ключей;
- вывод всего содержимого таблицы в прямом порядке первого ключа;
- вывод всего содержимого таблицы в диапазоне ключей без учета порядка их следования.
